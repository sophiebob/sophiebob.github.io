安装前提

安装 Hexo 非常简单。首先您必须安装下列软件：

Git
Node.js

安装Hexo

安装Node.js后,我们可以使用npm安装Hexo。

安装Hexo后,首先需要创建一个文件夹(F:\Hexo),在指定文件夹中执行下列命令(Hexo文件夹上右键，选择git bash), Hexo 将会在文件夹中创建所需文件。

等待安装,安装后可以看到许多文件。继续执行命令

这个时候，我们在浏览器中访问 http://localhost:4000/ ，就可以看到基于 Hexo 的默认主题的原型。

hexo常用命令

$ hexo g #完整命令为hexo generate,用于生成静态文件
$ hexo s #完整命令为hexo server,启动本地服务器
$ hexo d #完整命令为hexo deploy,用于文件发布到github上
$ hexo n #完整命令为hexo new,用于新建一篇文章

安装 next 主题

也可以直接到next上直接下载，放到theme目录下。

ps:
在 Hexo 中主要配置文件，其名为 _config.yml。我们称之为站点配置文件。
在主题themes目录下有第三方提供的主题配置文件\themes\next_config.yml。我们称之为主题配置文件。

使用 next 主题

打开 站点配置文件，找到 theme 字段，将其值更改为 next。

设置favicon图标

1.首先准备16X16和32X32大小的图片。可以去Easyicon上挑选。
2.在主题配置文件_config.yml，查找 favicon 修改配置文件

3.重新部署即可。

创建 分类、归档、关于

1.在主题配置文件_config.yml，查找 menu 修改配置文件

2.设置分类 新建一个页面，命名为categories

3.在Hexo/source目录下会生成categories目录

4.编辑categories文件夹下的index.md

5.同理设置标签

新建文章
