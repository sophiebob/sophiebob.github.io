阅读次数统计

next 也集成了 leancloud 。可以在leancloud进行账号注册。
创建一个新的应用。点击应用进入。
创建名称为Counter的Class。

点击设置 > 应用Key 复制App ID 和 App Key

修改配置文件

在主题themes目录下有第三方提供的主题配置文件\themes\next_config.yml。我们称之为主题配置文件。
打开主题配置文件 添加key

修改统计设置

打开主题配置文件 定位到 post_wordcount

Web安全性

为了保证应用的统计计数功能仅应用于自己的博客，你可以在应用 > 设置 > 安全中心的Web安全域名中加入自己的博客域名，保证数据的调用安全。
